# The Commons — Phase 2 Prototype

This folder contains a lightweight landing page prototype for the Phase 2 strategy: repositioning the public library as a warm, neighborhood-centered third space.

## What it includes

- A hero section framing the library as “Your place. Always open. Always free.”
- A neighborhood experience section for local programming and live updates
- A trusted messenger section for community partners
- A private digital on-ramp experience with expandable content

## Run locally

Open [index.html](index.html) in a browser, or serve the folder with a static server.
